#LaTeX-Converter
轻量级文本转LaTeX格式工具，一键将Markdown/纯文本转换为LaTeX可直接编译的.tex文件，支持打包为Windows可执行文件（.exe）。
## 项目介绍
本工具解决了手动编写LaTeX格式繁琐的问题，自动识别并转换常见文本格式（标题、列表、粗体、代码块、特殊字符转义等），输出的.tex文件可直接在TeXstudio、Overleaf等LaTeX软件中编译使用。核心特性：
- 格式全覆盖：支持标题、无序列表、粗体、代码块、特殊字符转义（&/~/%/$等）
- 中文友好：内置ctex包，完美支持中文编译
- 一键打包：提供完整流程将Python源码打包为Windows可执行文件（.exe）
- 轻量无依赖：核心功能仅依赖Python标准库，打包后单文件可独立运行
